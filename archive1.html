3. reservation inquiry service

This module is mainly used by the protocol fleet to query the reservation information based on the query criteria. If the reservation is not effective, the protocol fleet can delete and resubmit the reservation

Click on "Appointment Query", and without entering relevant information, click on "Query", and the system will perform a fuzzy search.

About This System

Our company's smart port system is a large project, so we adopt distributed architecture. Container reservation is one of the microservices in the smart port system.

Container reservation service refers to the process in which the cargo owner's fleet registers the container delivery appointment through the container reservation system before entering the corresponding port for container delivery.

Essential Tech Used In Project

  • The backend framework mainly uses Spring Cloud + Spring Boot
  • The registration center adopts Alibaba Nacos to complete the registration and discovery of services
  • Adopting Spring Gateway routing, Feign for remote calls between microservices, and Ribbon technology for load balancing calls between services
  • The Hystrix circuit breaker mechanism effectively prevents linkage failures in distributed systems, in order to improve the resilience of microservice systems
  • Spring validation is used to verify user submitted information. Spring declarative transaction management ensure the atomicity of operations on multiple database tables
